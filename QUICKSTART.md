# 🚀 БЫСТРЫЙ СТАРТ — MerAi & Monitoring v8.1

## ⚡ Запуск за 5 минут

### 1. Установка зависимостей

```bash
pip install -r requirements.txt --break-system-packages
```

**Важно:** Флаг `--break-system-packages` обязателен для BotHost.

### 2. Настройка токена бота

Откройте `bot.py` и замените:

```python
BOT_TOKEN = "8505484152:AAHXEFt0lyeMK5ZSJHRYpdPhhFJ0s142Bng"  # ← ВАШ ТОКЕН
```

### 3. Настройка канала верификации

Создайте приватный канал и получите его ID:

```python
CAPTCHA_CHANNEL = "https://t.me/+ВАШ_ПРИВАТНЫЙ_КАНАЛ"
CAPTCHA_CHAN_ID = -1001234567890  # ← ID канала
```

**Как получить ID канала:**
1. Добавьте бота в канал как администратора
2. Отправьте любое сообщение в канал
3. Используйте [@userinfobot](https://t.me/userinfobot) для получения ID

### 4. Настройка Admin ID

```python
ADMIN_ID = 7785371505  # ← ВАШ TELEGRAM ID
```

**Получить свой ID:** Напишите [@userinfobot](https://t.me/userinfobot)

### 5. (Опционально) API ключи Userbot

Если планируете userbot-функционал:

```python
PYRO_API_ID   = 12345678        # с my.telegram.org
PYRO_API_HASH = "abc123def456"  # с my.telegram.org
```

**Получение:**
1. Откройте https://my.telegram.org
2. Войдите под своим номером
3. API development tools → Create application
4. Скопируйте api_id и api_hash

### 6. Запуск бота

```bash
python3 bot.py
```

**Ожидаемый вывод:**
```
2026-03-10 12:00:00 | INFO     | MerAi | ✅ БД подключена (v8.1)
2026-03-10 12:00:01 | INFO     | MerAi | 🚀 Бот @ваш_бот (ID 123456) v8.1 стартует…
2026-03-10 12:00:02 | INFO     | MerAi | ✅ Запуск polling v8.1…
```

## 🔧 ПРОВЕРКА РАБОТОСПОСОБНОСТИ

### Тест 1: Базовый функционал

1. Напишите боту `/start`
2. Должно появиться приглашение к верификации
3. Вступите в канал через кнопку
4. Вернитесь к боту → нажмите `/start`
5. Должно появиться главное меню

### Тест 2: Тестовый период

После верификации проверьте:
- План: должен быть "🎁 Тестовый период"
- До: должна быть дата через 3 дня
- Все функции доступны

### Тест 3: Сохранение в ЛС

1. Отправьте боту любое сообщение (текст/фото)
2. Должно прийти уведомление "💾 Сообщение сохранено"
3. Проверьте профиль → статистика должна обновиться

### Тест 4: Команда "тест"

1. В любом чате напишите сообщение
2. Ответьте на него словом "тест"
3. Бот должен сохранить исходное сообщение
4. Команда "тест" должна удалиться

### Тест 5: Admin-панель

1. Напишите `/admin` (работает только для ADMIN_ID)
2. Должна открыться панель со статистикой
3. Проверьте раздел "Конфиг"
4. Попробуйте изменить test_period_days

## ⚙️ НАСТРОЙКА КОНФИГА (через /admin)

### Рекомендуемые значения для старта:

```
test_period_days: 3       # 3 дня пробного периода
clone_bonus_days: 3       # +3 дня за клонирование
ref_goal: 50              # 50 рефералов для награды
auto_verify_enabled: 1    # Автопроверка канала ВКЛ
channel_check_hours: 1    # Проверка каждый час
save_dm_messages: 1       # Сохранение ЛС ВКЛ
save_test_replies: 1      # Команда "тест" ВКЛ
```

### Оптимизация под нагрузку:

**Если пользователей < 100:**
- `channel_check_hours: 1` (каждый час)

**Если пользователей 100-1000:**
- `channel_check_hours: 3` (каждые 3 часа)

**Если пользователей > 1000:**
- `channel_check_hours: 6` (каждые 6 часов)
- Рассмотрите отключение `auto_verify_enabled: 0`

## 🐛 УСТРАНЕНИЕ ПРОБЛЕМ

### Проблема: "БД не подключена"

**Решение:**
```bash
# Проверьте права на запись
chmod 755 .
# Удалите старую БД (если есть)
rm merai.db
# Перезапустите
python3 bot.py
```

### Проблема: "Pyrogram не установлен"

**Решение:**
```bash
pip install Pyrogram==2.0.106 TgCrypto==1.2.5 --break-system-packages
```

### Проблема: "Token invalid"

**Решение:**
1. Проверьте формат токена: `123456789:ABC-DEF...`
2. Убедитесь, что токен активен (@BotFather → /mybots)
3. Проверьте отсутствие лишних пробелов

### Проблема: "Канал не найден"

**Решение:**
1. Убедитесь, что бот добавлен в канал как администратор
2. ID канала должен быть отрицательным: `-1001234567890`
3. Используйте полный числовой ID, а не username

### Проблема: "Userbot не запускается"

**Решение:**
1. Проверьте правильность `api_id` (число)
2. Проверьте `api_hash` (32 символа, a-f, 0-9)
3. Используйте inline-клавиатуру для ввода кода
4. Для 2FA введите пароль через кнопки

## 🔄 ОБНОВЛЕНИЕ С v8.0

Если у вас уже запущен v8.0:

1. **Остановите бота** (Ctrl+C)
2. **Сделайте резервную копию:**
   ```bash
   cp merai.db merai_backup.db
   cp bot.py bot_v8.0.py
   ```
3. **Замените файлы:**
   ```bash
   cp новый_bot.py bot.py
   cp новый_requirements.txt requirements.txt
   ```
4. **Обновите зависимости:**
   ```bash
   pip install -r requirements.txt --break-system-packages --upgrade
   ```
5. **Запустите бота:**
   ```bash
   python3 bot.py
   ```

**Автоматическая миграция:**
- Создадутся новые поля в БД
- Free → Trial конвертация
- Все платные подписки сохранятся
