## 🚀 Деплой на BotHost

### 1. Загрузи на GitHub

```bash
git init
git add merai_v7_bot.py requirements.txt
git commit -m "MerAi v7.0"
git push origin main
```

### 2. Настройка в коде (строки 45–55)

```python
BOT_TOKEN       = "ВАШ_ТОКЕН"          # от @BotFather
ADMIN_ID        = 7785371505           # твой Telegram ID
CAPTCHA_CHANNEL = "https://t.me/+..."  # приватный канал
CAPTCHA_CHAN_ID = -1003716882147       # ID канала (уже вставлен)
PYRO_API_ID     = 12345                # my.telegram.org/apps
PYRO_API_HASH   = "abc123..."          # my.telegram.org/apps
```

### 3. BotHost → команда запуска

```bash
pip install -r requirements.txt --break-system-packages && python3 merai_v7_bot.py
```

## ⚙️ Настройка капча-канала

1. Создай приватный канал
2. Добавь бота как **администратора** с правом **«Пригласить пользователей»**
3. Включи **«Одобрение заявок вручную»** в настройках канала  
4. Скопируй invite link → вставь в `CAPTCHA_CHANNEL`
5. ID канала уже вставлен (`-1003716882147`)

> **Важно:** бот НЕ одобряет заявки автоматически —  
> он лишь видит факт отправки заявки и разблокирует пользователя.

## 👤 Userbot Mode — получи API ключи

1. Открой [my.telegram.org/apps](https://my.telegram.org/apps)
2. Войди с номером телефона
3. Создай приложение → получи `api_id` и `api_hash`
4. Вставь в код (строки 52–53)
