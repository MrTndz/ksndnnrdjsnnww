## 🔧 РЕШЕНИЕ ПРОБЛЕМ

### Бот не запускается

```bash
pip install -r requirements.txt --break-system-packages
python3 --version  # Нужна 3.10+
```

### Капча не работает

1. Проверь `CAPTCHA_CHANNEL_ID = -1003716882147`
2. Бот должен быть админом в канале
3. Права: Ban users

### UserBot не запускается

1. Проверь session string
2. Pyrogram установлен?
3. Правильный формат session string

### Клоны не работают

- Токен правильный?
- Бот активен?
- Проверь логи
