## 📋 Описание

MerAi & Monitoring - это продвинутый Telegram бот с возможностью работы в двух режимах:

1. **Bot Mode** - стандартный бот для Telegram Business аккаунтов
2. **Userbot Mode** - полноценный userbot с доступом ко всем функциям

### Основные возможности

- 🔍 **Мониторинг удаленных сообщений** - перехват и сохранение удаленных сообщений
- 📝 **Отслеживание редактирований** - история изменений сообщений
- 📦 **Автоархивация** - создание ZIP архивов при массовом удалении (>10 сообщений за 3 сек)
- 🤖 **AI-ассистент** - интеграция с Gemini, Grok, GLM
- 💳 **Система подписок** - оплата через Telegram Stars, криптовалюту, банковские карты
- 🔗 **Поддержка пользовательских ботов** - подключение собственных ботов (+7 дней к подписке)
- 🎨 **Красивые уведомления** - HTML форматирование с эмодзи
- 👥 **Управление пользователями** - полная админ-панель

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt --break-system-packages
```

### 2. Настройка бота

#### Получение BOT_TOKEN:

1. Откройте https://t.me/BotFather
2. Создайте нового бота командой `/newbot`
3. Скопируйте токен

#### Для режима Userbot (опционально):

1. Откройте https://my.telegram.org/apps
2. Создайте новое приложение
3. Получите `API_ID` и `API_HASH`

**⚠️ ВАЖНО:** Каждый пользователь должен создать свое приложение! Одно приложение = один userbot.

### 3. Настройка переменных окружения

Создайте файл `.env`:

```bash
cp .env.example .env
nano .env
```

Минимальная конфигурация для запуска:

```env
BOT_TOKEN=ваш_токен_от_BotFather
MODE=bot
```

Для userbot режима:

```env
BOT_TOKEN=ваш_токен_от_BotFather
MODE=userbot
API_ID=ваш_api_id
API_HASH=ваш_api_hash
```

### 4. Запуск

```bash
python3 merai_bot.py
```

## 🤖 Режимы работы

### Bot Mode (Telegram Business)

**Преимущества:**
- ✅ Не требует API_ID/API_HASH
- ✅ Простая настройка
- ✅ Официально поддерживается Telegram

**Ограничения:**
- ❌ Работает только с Business аккаунтами (Telegram Premium)
- ❌ Перехват только в бизнес-чатах

**Настройка:**
1. Подключите Premium подписку
2. Telegram → Настройки → Telegram Business → Chatbots
3. Добавьте бота @ваш_бот

### Userbot Mode (Pyrogram)

**Преимущества:**
- ✅ Работает во всех чатах
- ✅ Полный доступ к API
- ✅ Перехват во всех типах чатов

**Ограничения:**
- ❌ Требует создания приложения на my.telegram.org
- ❌ Каждый пользователь должен создать свое приложение

**Настройка:**
1. Создайте приложение на https://my.telegram.org/apps
2. Получите API_ID и API_HASH
3. Добавьте в настройках бота `/settings`
4. Переключите режим на "Userbot"

## 🐛 Решение проблем

### Ошибка: "EOFError: EOF when reading a line"

**Причина:** Попытка использования `input()` в headless окружении

**Решение:** Убедитесь, что используете переменные окружения вместо интерактивного ввода. Все настройки должны быть в `.env` файле или в админ-панели.

### Ошибка: "Pyrogram not installed"

**Решение:**

```bash
pip install pyrogram tgcrypto --break-system-packages
```

### База данных не создается

**Решение:** Проверьте права доступа к папке:

```bash
chmod 755 .
```

### Бот не отвечает

1. Проверьте логи: `tail -f merai_bot.log`
2. Убедитесь что BOT_TOKEN корректный
3. Проверьте подключение к интернету
