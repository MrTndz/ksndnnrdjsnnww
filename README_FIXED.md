## 🔧 ОБРАБОТКА ОШИБОК

Все критические участки обернуты в try/except:

```python
try:
    # Опасная операция
except TelegramBadRequest as e:
    # Обработка ошибки Telegram
except TelegramForbiddenError as e:
    # Пользователь заблокировал бота
except Exception as e:
    # Общая обработка
    logger.error(f"Ошибка: {e}\n{traceback.format_exc()}")
```

## 🐛 РЕШЕНИЕ ПРОБЛЕМ

### Бот не запускается

```bash
# Проверить зависимости
pip install -r requirements.txt --break-system-packages

# Проверить Python
python3 --version  # Нужна 3.10+
```

### Капча не работает

1. Проверить `CAPTCHA_CHANNEL_ID` (должен быть `-1003716882147`)
2. Проверить что бот админ в канале
3. Права: Ban users

### UserBot не подключается

1. Проверить API_ID/API_HASH
2. Убедиться что Pyrogram установлен
3. Проверить номер телефона
